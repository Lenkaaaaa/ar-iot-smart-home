# AR IoT Smart Home

Diplomový projekt zameraný na ovládanie a monitorovanie IoT zariadení pomocou rozšírenej reality a 3D identifikátorov.

## Časti projektu
- `unity-app/` - AR aplikácia v Unity + Vuforia
- `raspberry-pi-server/` - serverová časť v Python
- `arduino/` - Arduino sketch pre senzory a akčné členy
- `docs/` - diagramy, screenshoty a implementačné poznámky

## Použité technológie
- Unity 2022.3
- Vuforia Engine
- Raspberry Pi 5
- Python
- WebSocket
- Flask / prípadne jednoduchý WebSocket server
- Arduino Uno
